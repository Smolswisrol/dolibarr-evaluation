# dolibarr-evaluation
# This repository does not include a full Dolibarr installation or live database dump. The official Dolibarr package should be downloaded from the official Dolibarr website (or through their repository). This repo contains project documentation, sample data, and installation notes for a Advanced Web Development final project.

Project assignment of an evaluation of the dolibarr application for a store's inventory and POS.

Reference Business: 
Barbour Farms Country Store in Byron, GA

Business Problem: 
Manual in-store and online inventory updates can cause inaccurate stock counts and extra work.

Application: 
Dolibarr ERP/CRM

Local Environment:
Windows, XAMPP, Apache, MySQL, PHP

Meain features demonstrated:
1. Product records with SKU/barcode values
2. Warehouse stock tracking
3. TakePOS sale reducing inventory
4. Future website/e-commerce synchronization discussion
